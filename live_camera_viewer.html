NYC DOT Live Camera Viewer

Fetches the current frame directly from webcams.nyctmc.org — no key needed. This only works as a local file / real webpage, not inside a claude.ai artifact (that sandbox blocks external image loads).

Loaded

Each refresh appends a cache-busting ?t= query param so the browser doesn't just show you a cached copy — the image itself still comes straight from the DOT camera API each time. The true frame timestamp is burned into the image by the camera; "last refreshed" below is when your browser requested it.
